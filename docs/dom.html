Dom Manipulation

Getting Started

With Mei.js we have two simple methods to use. Both have the option to handle a function or just save the element in a simple variable. In both methods you can pass any css selector to access him. Let's take a look!

getElement:

Return a specific element

Mei.getElement('element'); 


  const element = Mei.getElement('#myelement');

getElement with an event handler:

set an event to an element.

Mei.getElement('element', 'event', callback) 


  Mei.getElement('#myelement','click', () => {
    // code
  });

getElements:

Return an array of elements.

Mei.getElements('elements'); 


  const elements = Mei.getElements('#myelements');

getElements with an event handler:

Set the same event for every element selected. This practice is Not so recommended. Use event delegation instead.

Mei.getElements('elements', 'event', callback) 


  Mei.getElements('#myelements','click', () => {
      // code
  });
