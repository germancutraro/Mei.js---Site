Changing the DOM

Getting Started

Mei have three methods to use one for create elements, one for delete a specific element an the last one is for clone a element. Let's take a look!

createElement:

create a element in the document. You can specified any valid attribute.

Mei.createElement('element', {attributes}); 


  // This will create a 'h1' element
  Mei.createElement('h1', {
    textContent: 'Hello World!',
    className: 'title',
    id: 'title-js',
    parent: document.body
  });

    // This will create a 'a' element
  Mei.createElement('a', {
    textContent: 'Mei.js',
    href: 'https://github.com/germancutraro',
    target: '_blank',
    parent: document.body
  });

  // Creating Childs
  Mei.createElement('ul', {
    id: 'list-item',
    parent: document.body,
    children: [{
      element: 'li',
      textContent: 'Item-1',
      class: 'item'
    }]
  });
parent: You can specified any existing element here.
children: You must pass an array of objects. The parent will be the first element created.

removeElement:

delete the last element as a parameter.

Mei.removeElement('element'); 


  // This will remove the last 'ul' that we created.
  Mei.removeElement('#list-item'); 

  // Or you can remove a element when you are selecting it.
  Mei.removeElement(Mei.getElement('#list-item'));

cloneElement:

Copy a element.

Mei.cloneElement('element'); 


  // Cloning the 'h1' that we created
  let h1 = Mei.getElement('.title'); 
  let newH1 = Mei.cloneElement('.title'); 

  // printing out
  console.log(newH1);
