Storage

Getting Started

The storage is a great feature that we have. With Mei you are going to be more easy! Let's see

store:

This method defines the type of store and you can define the item to save.

Mei.store({attributes}); 


  let users = [{name: 'John'}, {name: 'Nick'}];
  // This will create a 'users' item in a localStorage way.      
  Mei.store({
    store: 'local',
    item: 'users',
    data: users
  });

  // This will create a 'users' item in a sessionStorage way.
  Mei.store({
    store: 'session',
    item: 'users',
    data: users
  });

clearStore:

This will clear all records saved according to the type of storage.

Mei.clearStore('type'); 


  // This will clear all records saved in the localStorage.
  Mei.clearStore('local');

  // This will clear all records saved in the sessionStorage.
  Mei.clearStore('session');

removeItemStore:

This will clear all records saved according to the type of storage.

Mei.removeItemStore('type', 'item'); 


  // This will clear all records saved in the localStorage.
  Mei.removeItemStore('local');
    
  // This will clear all records saved in the sessionStorage.
  Mei.removeItemStore('session');

displayStore:

.

Mei.displayStore('type', 'item'); 


  // outputting all the records that match with the 'users' localStorage item.
  console.log(Mei.displayStore('local', 'users'));

  // outputting all the records that match with the 'users' sessionStorage item.
  console.log(Mei.displayStore('session', 'users'));
